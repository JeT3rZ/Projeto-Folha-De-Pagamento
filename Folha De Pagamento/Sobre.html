O que é folha de pagamento ?

A folha de pagamento é um documento que contém o registro detalhado de todos os vencimentos e descontos devidos aos funcionários de uma empresa em determinado período. Esse documento é essencial para calcular e distribuir corretamente os salários dos empregados, além de cumprir com as obrigações fiscais e trabalhistas

Foi Desenvolvido, ao longo do semestre, a simulação de um site de Folha de Pagamento. O projeto tem como objetivo aplicar os conhecimentos adquiridos em sala de aula, promovendo a integração prática dos conceitos de desenvolvimento web, lógica de programação e gestão de dados.
A simulação irá representar as principais funcionalidades de um sistema de folha de pagamento, como:
  • Cadastro de funcionários;
  • Cálculo de salários, descontos e benefícios;
  • Geração de contracheque;
  • Gerenciamento de usuários, com a possibilidade de edição e exclusão de perfis pelo administrador;
  • Separação de tipos de usuários cada um com permissões e funcionalidades específicas de acordo com seu nível de acesso.
Para o desenvolvimento do sistema, foram utilizadas as seguintes tecnologias:
  • HTML para a estruturação das páginas;
  • PHP para a lógica de programação e integração com o servidor;
  • MySQL como sistema de gerenciamento de banco de dados, responsável pelo armazenamento e consulta das informações.
